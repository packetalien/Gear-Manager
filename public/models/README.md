# 3D Models Directory

Place the `mannequin.glb` file here.

**Required Asset:**
- File: `mannequin.glb`
- Source: CC0 Prototyping Mannequin
- Format: GLTF Binary (.glb)
- Recommended: ~9.6k tris, neutral pose, two materials

The model will be automatically loaded by the PaperDoll component using React Three Fiber.
